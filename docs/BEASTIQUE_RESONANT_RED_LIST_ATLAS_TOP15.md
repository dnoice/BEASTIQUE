# BEASTIQUE — Resonant Red List Atlas

## Curated Top 15 Endangered Species That Most Powerfully Resonate with Humans

## 1. Purpose

This artifact establishes a **curated flagship species list** for the **BEASTIQUE Project**. It is not intended to be a mechanically literal ranking of the 15 globally rarest taxa by absolute population alone. Instead, it is a **human-resonance-weighted conservation shortlist** built around the BEASTIQUE mission:

* highlight species facing severe extinction pressure,
* preserve scientific seriousness,
* maximize emotional and narrative connection,
* generate visually compelling species candidates for BEASTIQUE’s material-fusion image system,
* create a reusable prompt library for production image generation.

Each selected species includes:

1. **Species identity block**
2. **Why it resonates with humans**
3. **A proposed BEASTIQUE material pairing**
4. **12 fascinating facts**
5. **12 corresponding 1:1 image prompts**, each engineered for a modern AI image pipeline

---

## 2. Important Curation Note

This list should be understood as a **BEASTIQUE editorial shortlist**, not a claim that these are the only endangered species that matter, nor a hard scientific assertion that they are the 15 most endangered organisms on Earth across all taxa.

### Core curation filters:

* Extremely high conservation urgency
* Strong human emotional resonance
* High narrative power
* Strong visual identity
* Taxonomic and ecological spread
* Strong BEASTIQUE image potential

### Source-of-truth rule:

Final IUCN category labels, population language, and geography should be **hydrated from the project’s internal latest Red List dataset** before publication.
